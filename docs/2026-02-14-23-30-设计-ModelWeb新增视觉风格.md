# 设计 - Model Web 新增视觉风格扩展

## 1. 背景 (Background)
用户希望在 `Model Web` 网页生成助手（Web Architect）中提供更多元化的视觉选择。目前系统仅支持 Swiss International, Warm Minimalist, 和 Neo-Brutalist 三种风格。为了提升产品的演示效果和生成的丰富度，我们需要新增三种现代流行的视觉风格。

## 2. 目标 (Goals)
- 在 `Web Architect` 助手中集成三种新视觉风格：Glassmorphism, Cyber-Glow, 和 Bento Modern。
- 确保新增风格在 UI 侧正确展示并可被用户选择。
- 确保 Agent 能够理解新风格的视觉语义并生成符合规范的代码。

## 3. 方案设计 (Design Specification)

### 3.1 风格定义详情

#### 1. Glassmorphism (玻璃拟态)
- **ID**: `glassmorphism`
- **视觉特征**: 
    - **整体**: 现代、轻量、通透。
    - **色彩**: 柔和的浅色或鲜艳渐变背景，卡片使用半透明白 (`bg-white/30`) 或黑 (`bg-black/30`)。
    - **形状**: 适中圆角 (`rounded-2xl`)，强调 `backdrop-blur-md`。
    - **边框**: 极细的白色半透明边框 (`border border-white/20`) 模拟玻璃边缘反光。
- **字体**: 清晰的无衬线字体 (如 Inter)。

#### 2. Cyber-Glow (赛博霓虹)
- **ID**: `cyber-glow`
- **视觉特征**: 
    - **整体**: 暗黑模式、未来感、高能量。
    - **色彩**: 深色背景 (`#0F172A`)，霓虹青 (`#22D3EE`) 或品红 (`#F02AD3`) 作为强调色。
    - **形状**: 较小圆角 (`rounded-md`)。
    - **特效**: 文字和边框带有发光效果 (`drop-shadow-[0_0_8px_rgba(34,211,238,0.8)]`)。
- **字体**: 具有技术感的等宽字体 (如 Space Mono) 或现代无衬线字体。

#### 3. Bento Modern (现代便当盒)
- **ID**: `bento`
- **视觉特征**: 
    - **整体**: 结构化、模块化、Apple 风格。
    - **色彩**: 极简浅灰背景 (`#F8FAFC`)，白色卡片，深灰色文字。
    - **形状**: 大圆角卡片 (`rounded-3xl`)。
    - **布局**: 严格的网格系统，内容分块明确。
- **字体**: 优雅、间距考究的无衬线字体 (如 Inter 或 SF Pro 风格)。

### 3.2 实现路径

1.  **后端 Prompt 更新**: 在 `apps/studio/src/assistants/web-architect/prompts/designs.ts` 中添加新的配置对象。
2.  **前端 UI 更新**: 
    - 在 `apps/studio/src/components/web/web-config.tsx` 中导入并映射新的 Lucide 图标。
    - 适配新增风格的图标展示。

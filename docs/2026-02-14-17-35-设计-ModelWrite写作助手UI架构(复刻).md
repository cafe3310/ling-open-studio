# 2026-02-11-00-04-设计-ModelWrite写作助手UI架构

## 1. 背景 (Background)

基于 InclusionAI 设计系统和产品原型，我们需要开发 "Model Write" 助手。这是一个专注于创意写作、具 备 AI 辅助灵感生成能力的沉浸式编辑器。

## 2. 视觉规范 (Visual Specs)

### 2.1 配色方案 (Color Palette)
- **背景**: Ice Gray (#F8F9FB) 作为侧边栏背景，纯白 (#FFFFFF) 作为写作区背景。
- **文字**: Brand Dark (#1A1A1A) 用于标题和正文，Playfair Display 字体。
- **装饰**: Action Blue (#3B82F6) 用于主按钮，Tech Cyan 用于次要标签。

### 2.2 布局结构 (Layout)
- **三栏式隔离布局**:
    - **左侧 (280px)**: `Project Metadata`。包含标题、摘要、标签组、角色列表。
    - **中间 (Flexible)**: `Writing Canvas`。模拟纸张体验，最大宽度限制（如 800px），居中显示。
    - **右侧 (320px)**: `Intelligence Sidebar`。灵感卡片流 (Inspiration Cards)。

## 3. 组件拆分 (Component Breakdown)

- `src/components/assistants/write-architect/`:
    - `index.tsx`: 助手入口，管理布局。
    - `write-left-sidebar.tsx`: 静态设置与角色卡片。
    - `write-canvas.tsx`: 编辑器容器。
    - `write-right-sidebar.tsx`: 灵感卡片展示。

## 4. 后续功能预留 (Future Functions)

- 基于 VFS 的文档自动保存。
- LangGraph 驱动的“续写下一段”和“灵感生成”流。
- 角色与世界观的上下文注入。
